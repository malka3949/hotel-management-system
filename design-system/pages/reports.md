### KPI Cards

- Grid: `grid grid-cols-2 md:grid-cols-4 gap-4`
- כל כרטיס: ערך ראשי `text-4xl font-bold` + אחוז שינוי + trend arrow
- trend arrow: `TrendingUp` (ירוק) / `TrendingDown` (אדום) / `Minus` (אפור)

### פילטרים

- **תאריך**: ברירת מחדל = 30 יום אחרונים. אפשרויות: היום / שבוע / חודש / Custom range
- **סניף** (chain_admin בלבד): dropdown עם כל הסניפים + "כל הסניפים"

### גרפים

- ספריה: shadcn/ui charts (Recharts based)
- גרף תפוסה: `AreaChart` — ציר Y = % תפוסה, ציר X = תאריכים
- גרף הכנסות: `BarChart` — ציר Y = ₪, ציר X = תאריכים
- גרפים: `h-64` (256px), `w-full`
- Loading: skeleton `h-64 animate-pulse`

### טבלת השוואת סניפים (chain_admin בלבד)

- ממוין לפי תפוסה (ברירת מחדל)
- לחיצה על שורה → מסנן את כל הדוח לסניף הנבחר

## Payment Reconciliation (`/reports/reconciliation`)

```
┌──────────────────────────────────────────────────────────────┐
│ פיוס תשלומים                   [תאריך ▾] [ייצוא CSV]          │
├──────────────────────────────────────────────────────────────┤
│ [KPI] סה"כ חויב  [KPI] סה"כ שולם  [KPI] יתרה פתוחה  [KPI] החזרות │
├──────────────────────────────────────────────────────────────┤
│ [סטטוס ▾]  [שיטת תשלום ▾]                                    │
├────────────────────────────────────────────────────────────── │
│ [טבלת תשלומים]                                                │
│  תאריך │ הזמנה │ אורח │ סכום │ שיטה │ סטטוס │ ספק            │
└──────────────────────────────────────────────────────────────┘
```

- "ייצוא CSV" — מוריד את הנתונים המסוננים כ-CSV
- כל שורה: לחיצה → `/invoices/[id]`
- סטטוס תשלום: badge צבעוני (succeeded=ירוק, failed=אדום, pending=amber, refunded=אפור)

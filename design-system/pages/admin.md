# Page Override: Admin (`/admin/branches`, `/admin/users`)

> Overrides MASTER.md for admin management pages.
> Access: chain_admin only (both pages). hotel_manager: /admin/users (own branch only).

---

## Branch Management (`/admin/branches`)

```
┌──────────────────────────────────────────────────────────────┐
│ ניהול סניפים                              [+ סניף חדש]       │
├──────────────────────────────────────────────────────────────┤
│ [חיפוש: שם סניף]                          [X סניפים]        │
├──────────────┬────────────┬──────────┬────────┬──────────────┤
│ שם סניף      │ כתובת      │ טלפון   │ פעיל   │ פעולות       │
├──────────────┼────────────┼──────────┼────────┼──────────────┤
│ ת"א מרכז     │ ...        │ ...      │ ✓      │ [ערוך]       │
└──────────────┴────────────┴──────────┴────────┴──────────────┘
```

### פעולות

- "סניף חדש" → Dialog `max-w-md`: שם, כתובת, טלפון, אימייל, איש קשר, timezone
- "ערוך" → Dialog עם אותם שדות, מולא מראש
- השבתת סניף: toggle "פעיל" inline — AlertDialog "להשבית סניף X? כל המשתמשים שלו לא יוכלו להתחבר" → אשר

---

## User Management (`/admin/users`)

```
┌──────────────────────────────────────────────────────────────┐
│ ניהול משתמשים                             [+ משתמש חדש]      │
├──────────────────────────────────────────────────────────────┤
│ [חיפוש: שם / אימייל]  [תפקיד ▾]  [סניף ▾ — chain_admin]    │
├──────────┬──────────┬──────────┬──────────┬──────────────────┤
│ שם       │ אימייל   │ תפקיד   │ סניף     │ פעולות           │
└──────────┴──────────┴──────────┴──────────┴──────────────────┘
```

### תפקיד badges

| תפקיד | Badge |
|-------|-------|
| chain_admin | `bg-primary text-white` |
| hotel_manager | `bg-indigo-100 text-indigo-800` |
| receptionist | `bg-blue-100 text-blue-800` |
| housekeeping | `bg-amber-100 text-amber-800` |

### פעולות

- "משתמש חדש" → Dialog `max-w-md`: שם, אימייל, סיסמה זמנית, תפקיד, סניף
  - סיסמה זמנית: generated automatically, מוצגת פעם אחת — "העתק סיסמה"
  - chain_admin: ניתן לבחור כל סניף
  - hotel_manager: ניתן ליצור רק משתמשים בסניף שלו
- "ערוך" → Dialog: שם, תפקיד (chain_admin בלבד יכול לשנות), סניף
- "השבת" → AlertDialog → משתמש לא יכול להתחבר (is_active = false)
- אין מחיקת משתמש — השבתה בלבד

### הרשאות

| פעולה | chain_admin | hotel_manager |
|-------|-------------|---------------|
| צפייה בכל המשתמשים | ✓ | ✗ (סניף שלו בלבד) |
| יצירת משתמש | ✓ | ✓ (בסניף שלו) |
| שינוי תפקיד | ✓ | ✗ |
| שינוי סניף | ✓ | ✗ |
| השבתה | ✓ | ✓ (בסניף שלו) |

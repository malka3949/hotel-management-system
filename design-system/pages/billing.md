## Payment Checkout (`/payments/checkout`)

> נפתח מ-"גבה תשלום" בחשבונית או בהזמנה.

```
┌──────────────────────────────────────────────────────────────┐
│ גביית תשלום                                                   │
│ הזמנה #RES-0123 — ישראל ישראלי                               │
├──────────────────────────────────────────────────────────────┤
│ סיכום חשבונית                                                 │
│  לינה: ₪1,500 | מיניבר: ₪45 | סה"כ: ₪1,545                  │
│  שולם: ₪500 | **יתרה לתשלום: ₪1,045**                        │
├──────────────────────────────────────────────────────────────┤
│ שיטת תשלום                                                    │
│  ○ כרטיס אשראי    ○ מזומן    ○ מסוף POS                      │
│                                                               │
│  [כרטיס: Stripe Elements card input]                         │
│  [מזומן: ₪ input "סכום שהתקבל"]                              │
│  [POS: כפתור "חייב מסוף" + spinner]                          │
│                                                               │
│  [שלם ₪1,045]  ← disabled עד form valid                      │
└──────────────────────────────────────────────────────────────┘
```

- Stripe Elements: iframe מ-Stripe, לא custom card inputs
- הצלחה → redirect חזרה ל-`/invoices/[id]` + toast "תשלום התקבל"
- כישלון → error toast + כפתור חוזר לפעיל (לא navigate away)

## Guests List (`/guests`)

```
┌──────────────────────────────────────────────────────────────┐
│ אורחים                                      [+ אורח חדש]     │
├──────────────────────────────────────────────────────────────┤
│ [חיפוש: שם / טלפון / מספר דרכון]        [12 תוצאות]         │
├──────────┬────────────┬──────────┬───────────────────────────┤
│ שם מלא  │ טלפון      │ אימייל   │ הזמנות  │ פעולות          │
├──────────┼────────────┼──────────┼──────────┼─────────────────┤
│ ...      │ ...        │ ...      │ [count]  │ [הצג] [ערוך]   │
└──────────┴────────────┴──────────┴──────────┴─────────────────┘
```

- לחיצה על שורה → `/guests/[id]`
- "אורח חדש" → Dialog `max-w-md` (שם, טלפון, אימייל, מספר דרכון)
- חיפוש: debounce 300ms, חיפוש לפי שם / טלפון / מספר דרכון
- עמודת "הזמנות": count badge עם לינק לרשימת הזמנות האורח

## Guest Detail (`/guests/[id]`)

```
┌──────────────────────────────────────────────────────────────┐
│ [← חזרה]  ישראל ישראלי                      [ערוך]          │
├──────────────────────────────┬───────────────────────────────┤
│ פרטים אישיים                 │ היסטוריית הזמנות              │
│  שם מלא: ישראל ישראלי        │ [טבלת הזמנות האורח]           │
│  טלפון: 050-...              │  (check-in, check-out,        │
│  אימייל: ...                 │   סטטוס, חדר)                 │
│  דרכון: ...                  │                               │
│  נוצר: 01/01/2026            │                               │
└──────────────────────────────┴───────────────────────────────┘
```

- 2 עמודות: פרטים (1/3) + היסטוריה (2/3)
- "ערוך" → Dialog `max-w-md` עם React Hook Form
- ניתן לערוך: שם, טלפון, אימייל, מספר דרכון בלבד
- מחיקת אורח: אסור אם קיימות הזמנות פעילות (כפתור disabled + tooltip)

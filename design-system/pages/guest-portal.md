# Page Override: Guest Portal (`/portal/*`)

> Public-facing pages — no sidebar, no topbar, no staff navigation.
> All pages accessible via tokenized link only (no guest login).
> Layout: centered, `max-w-lg mx-auto`, clean and minimal.

---

## Portal Layout (all portal pages)

```
┌──────────────────────────────────────────────────────────────┐
│              [לוגו מלון]   [שם מלון / סניף]                  │
├──────────────────────────────────────────────────────────────┤
│                    [תוכן הדף]                                 │
├──────────────────────────────────────────────────────────────┤
│          טלפון מלון: 03-XXX-XXXX  |  support@hotel.co.il    │
└──────────────────────────────────────────────────────────────┘
```

- רקע: `bg-base`
- תוכן: `bg-surface rounded-xl shadow-md p-6`
- footer: תמיד מוצג — טלפון + אימייל לסיוע

## Reservation Landing (`/portal/[token]`)

```
┌──────────────────────────────────────────────────────────────┐
│ שלום, ישראל ישראלי                                            │
│ ─────────────────────────────────────────────────────────────│
│ הזמנה מספר: #RES-0123                                        │
│ סניף: מלון ים התיכון, תל אביב                                │
│                                                               │
│ ┌────────────────────────────────────────────────────────┐   │
│ │  📅 כניסה: ראשון 15/01/2026  יציאה: רביעי 18/01/2026  │   │
│ │  🛏  חדר 205 — חדר זוגי      3 לילות                   │   │
│ │  💰 סה"כ: ₪1,500  |  סטטוס: [שמור]                    │   │
│ └────────────────────────────────────────────────────────┘   │
│                                                               │
│  [צ'ק-אין אונליין]  [שלם עכשיו]  [הורד חשבונית]            │
└──────────────────────────────────────────────────────────────┘
```

**כפתורים — מצב לפי מצב הזמנה:**

| מצב | צ'ק-אין אונליין | שלם עכשיו | הורד חשבונית |
|-----|-----------------|-----------|--------------|
| reserved (לפני 24ש לכניסה) | disabled + tooltip | enabled (אם חוב) | enabled (אם קיימת) |
| reserved (תוך 24ש) | enabled | enabled | enabled |
| checked_in | disabled "כבר בוצע" | enabled | enabled |
| checked_out | disabled | disabled | enabled |

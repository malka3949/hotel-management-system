# Page Override: Auth (`/login`)

> Overrides MASTER.md for authentication pages.
> Public routes — no sidebar/topbar.

---

## Login Page (`/login`)

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                    [לוגו מלון]                               │
│                  מערכת ניהול מלון                            │
│                                                              │
│          ┌────────────────────────────────────┐              │
│          │  אימייל *                           │              │
│          │  [input]                            │              │
│          │                                     │              │
│          │  סיסמה *                            │              │
│          │  [input type=password]  [👁]        │              │
│          │                                     │              │
│          │  [כניסה למערכת]  ← accent button    │              │
│          └────────────────────────────────────┘              │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### עיצוב

- רקע: `bg-base` (`#F8FAFC`) — אותו רקע כמו האפליקציה
- כרטיס טופס: `bg-surface`, `rounded-xl`, `shadow-md`, `p-8`, `max-w-sm`, `mx-auto`
- מרכז אנכי: `min-h-screen flex items-center justify-center`
- כפתור "כניסה": `accent` (`#CA8A04`) — CTA יחיד בדף
- שגיאת כניסה (אימייל/סיסמה שגויים): toast error + שדות לא מאופסים

### ולידציה

- אימייל: format validation + required
- סיסמה: required, min 8 תווים (לא מציג דרישות — רק שגיאה אם קצרה מדי)
- שגיאות: inline מתחת לשדה, בעברית
- Submit disabled בזמן loading

### אחרי כניסה מוצלחת

- redirect → `/dashboard`
- אם קיים `?redirect=` param → redirect לנתיב הנדרש

### טיפול שגיאות

| שגיאה | הודעה |
|-------|-------|
| אימייל/סיסמה שגויים | "פרטי הכניסה שגויים" |
| משתמש לא פעיל | "חשבון זה אינו פעיל — פנה למנהל" |
| שגיאת שרת | "שגיאת מערכת — אנא נסה שנית" |

### נגישות

- autofocus על שדה אימייל
- `Enter` מגיש את הטופס
- `aria-label` על כפתור הצגת/הסתרת סיסמה
